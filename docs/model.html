Methodology

Model Specification

This analysis uses Ordinary Least Squares (OLS) regression to model county-level voter turnout rates as a function of five structural predictors from the American Community Survey. The unit of analysis is the county; the outcome variable is the county-level turnout rate calculated using CVAP denominators.

The model takes the form:

\[\text{Turnout}_i = \beta_0 + \beta_1(\text{Bachelor's}_i) + \beta_2(\text{Internet}_i) + \beta_3(\text{LEP}_i) + \beta_4(\text{Disability}_i) + \beta_5(\text{Poverty}_i) + \varepsilon_i\]

where \(i\) indexes counties and \(\varepsilon_i\) is the residual — the quantity of primary analytic interest.

Why OLS? The goal of this model is not prediction accuracy per se, but residual interpretation. OLS produces easily interpretable residuals in the original units (percentage points of turnout) and makes assumptions transparent. The model is intentionally parsimonious — omitted variable bias is expected, and its signature in the residuals is the point of the qualitative analysis.

Limitations This model is cross-sectional and observational. Coefficients should not be interpreted as causal effects. OLS assumes linearity, homoscedasticity, and no perfect multicollinearity; these assumptions are assessed but not guaranteed to hold across all counties. The model uses 2024 ACS data as a proxy for 2020 structural conditions; temporal mismatch may introduce measurement error for rapidly changing counties.

Model estimated in R using base lm(). Confidence intervals via broom::tidy(). Visualizations with ggplot2. Full replication code on GitHub.
